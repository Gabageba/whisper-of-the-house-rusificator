# Whisper of the House — Русификатор

Русская локализации игры **Whisper of the House** (нейро). 

## Дополнительно
Накиданы всякие инструменты, которые я использовал для работы с файлами локализации(многие могут быть устревшими). Вообщем помойка)

## Структура проекта

| Файл | Описание |
|------|----------|
| `I2Languages.json` | Файл локализации игры (нужно положить в папку проекта вручную) |
| `extract_english_keys.py` | Извлекает ключи и английский текст из I2Languages.json |
| `build_russian_keys.py` | Собирает итоговый файл с русскими переводами |
| `russian_translations.py` | Словарь естественных русских переводов (квесты, локации, персонажи, подсказки и т.д.) |
| `item_name_glossary.py` | Глоссарий переводов названий предметов (ItemName) |
| `english_keys.json` | Результат извлечения: ключ → английский текст (генерируется) |
| `russian_keys.json` | Итоговый файл: ключ → русский текст (генерируется) |

## Как использовать

### 1. Подготовка

Скопируйте файл **I2Languages.json** из папки с игрой в корень этого проекта.

### 2. Извлечение английских ключей

Извлечь все ключи и английские строки из I2Languages.json в `english_keys.json`:

**Windows (батник):**
```bat
run_extract_english_keys.bat
```

**Или вручную:**
```bash
python -u extract_english_keys.py
```

### 3. Сборка русской локализации

Собрать `russian_keys.json` из `english_keys.json`, подставив переводы из `russian_translations.py` и `item_name_glossary.py`:

```bash
python -u build_russian_keys.py
```

Для ключей, для которых нет готового перевода, в результат попадает исходный английский текст (их можно перевести позже отдельно).

### 4. Использование результата

Файл **russian_keys.json** — это словарь «ключ → русский текст». Его нужно подставить обратно в формат игры (например, вставить русские строки в нужное место структуры I2Languages или использовать в своём патче/моде).

## Какие ключи переводятся

Готовые русские переводы подставляются для категорий:

- **QuestCompletionPrompt**, **LevelQuestInfo**, **QuestDescription2**, **QuestLetter** — квесты
- **CharacterName** — имена персонажей
- **Address** — названия локаций и комнат
- **Image**, **Tip**, **QuestTip**, **Mail** — подсказки и письма
- **ItemInfo**, **ItemName** — описания и названия предметов (в т.ч. через глоссарий)
- **Dialogues** — диалоги
- и другие, добавленные в `russian_translations.py` и `item_name_glossary.py`

## Добавление переводов

- Общие тексты (квесты, локации, персонажи и т.д.) — в **russian_translations.py** (словари и функции в `get_all_natural_translations()`).
- Названия предметов — в **item_name_glossary.py** (словари `PHRASES` и при необходимости другие).

После правок достаточно заново запустить `build_russian_keys.py`.

## Кодировка

Все скрипты работают в UTF-8; JSON сохраняется с `ensure_ascii=False` для читаемых русских символов в файлах.
