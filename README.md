# Whisper of the House — Русификатор (нейро)

Русская локализация игры **Whisper of the House** (нейро).

## Дополнительно

В репозитории лежат инструменты, которые использовались для работы с файлами локализации (многие могут быть устаревшими). В общем, помойка :)

## Структура проекта

| Путь | Описание |
|------|----------|
| `extract_english_keys/` | Тут лежит скрипт для извлечения пары "ключ: английский текст" из I2Languages.json |
| `build_russian_keys/` | Пытался сделать перевод на основе словаря, получилась шляпа, решил каждую строку переводить вручную |
| `localization/` | Тут лежат файлы с английскими и русскими ключами локализации (тут все актуально для версии сборки)|
| `I2Languages.json` | Собственно таблица ключей, вытащенная из resources.assets|
